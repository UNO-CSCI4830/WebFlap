# WebFlap

## Summary

Welcome to WebFlap, a web-based tool based on the Java app JFLAP used in UNO's Theory of Computation course.

## Installation/deployment instructions for Mac

### First-time setup

- Install homebrew (https://brew.sh/)
- brew install npm

### All app runs (first-time and others)

- npm install (at project root directory, if package changes have occurred)
- cd webflap
- npm run dev
